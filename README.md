# Webbybot

[![Build Status](https://travis-ci.org/gasolin/webby.png)](https://travis-ci.org/gasolin/webbybot) [![Dependency Status](https://david-dm.org/gasolin/webby/dev-status.svg)](https://david-dm.org/gasolin/webbybot)

## Setup Development

```
$ npm install -g mocha
```

## Build

run command

```
$ npm run build
```

## Run

run command

```
$ node ./bin/webby.js
webby >
```

## Test

```
$ npm test
```

## Lint
```
$ npm run lint
```
